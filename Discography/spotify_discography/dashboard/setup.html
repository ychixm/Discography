Avant de commencer, créez une application Spotify pour obtenir vos identifiants OAuth.

  1. Allez sur developer.spotify.com/dashboard
  2. Cliquez Create app
  3. Renseignez un nom et une description quelconques
  4. Dans Redirect URIs, ajoutez : http://127.0.0.1:8888/callback
  5. Copiez le Client ID et le Client Secret
Doit correspondre exactement à ce que vous avez saisi dans le Dashboard Spotify.
